Nesta seção devem ser feitas as definições gerais do modelo:

1.       Local onde os arquivos serão salvos.

2.       Arquivo (.shp) com os dados espaciais.

3.        Nome do modelo.

4.       Ano de início do modelo.

5.       Ano de término do modelo.

 

                                                                                       

                                                             Figura 1: Definições do Modelo

 

 

 

A pasta onde o modelo será salvo pode ser acessada ao clicar em Selecionar em frente a "Pasta para Salvar o Modelo",  onde, uma janela de escolha de pasta será aberta:

 

 

                                  

                                            Figura 2: Janela para escolha da pasta de destino

 

 

 

 

 

Após a seleção da pasta aonde o Arquivo será salvo, o endereço selecionado será exibido para o usuário:

 

  

                         Figura 3: Exibindo caminho selecionado (Pasta para o Modelo)

                                            Figura 4: Janela para escolha do arquivo shape
