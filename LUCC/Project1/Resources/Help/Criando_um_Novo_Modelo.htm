There are two ways to access the form for creating a new template:

  1. Using the New Model button:

                     

                                                      Figure 1: New Model Button

 

2.       Through the Main Menu:

 

                       

                                         Figure 2: Main Menu - File > New

 

 

Once selected, the model creation window opens with some default values that must be changed to suit the application you want to create.

These values are just to give the user an idea of what type of data to contain in each field. If you want the user can use these default values.

 

 

                                                         Figure 3: Creating New Model Window

 

As you can see the model creation window is divided into parts (tabs), according to the types of data that must be present in the model.
