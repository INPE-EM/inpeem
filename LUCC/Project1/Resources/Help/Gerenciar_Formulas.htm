The formulas to be used in the model can be managed through the Equations Manager window:

                             Figure 1: Equations Manager Window

 

The used equations are stored in the tool and are available to be selected on all models.

 

If a model is loaded and its equations are not listed in the tool, they will be loaded automatically.

 

To access the Equations Manager, simply click on the Equations Manager menu:
