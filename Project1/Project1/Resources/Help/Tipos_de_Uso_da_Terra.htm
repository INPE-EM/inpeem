In this section, you must inform the land use types that you will use in your application.

The static types should also be informed as Types of Land Use No Data.

                                                                       Figure 1: Land Use Types

 

In order to add the Land Use Types the user must click in Manager in front of Land Use Types.

 

                                                       Figure 2: Manager Button – Land Use Types

 

 

After click the Land Use Types management window will appears:

 

 

                        Figure 3: Land Use Types Management window

 

 

The Land Use Type can be:

1.       Add

2.       Remove

3.       Order

4.       Save

 

 

                             Figure 4: Land Use Types Management

 

Note that some features are only visible when a Land Use Type is selected (difference between Figures 3 and 4) .

After save the data a confirmation appears to the user:

                                                     Figure 5: Land Use Types confirmation

 

In order to add the Land Use No Data, which Land Use Type will receive no data (e.g. border data), the user must click in Manager in front of Land Use No Data.

Note that the Land Use Types will come pre-populated if the Land Use Types are already defined, the user just simply selecting the desire land use type.

 

                                                   Figure 6: Manager Button – Land Use No Data

 

                           Figure 7: Land Use No Data Window

 

After selecting the desired land use a confirmation will appear to the user:

 

                                                    Figure 8: Land Use No Data Confirmation
