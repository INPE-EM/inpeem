In this section, the data from a Spatial Model must be defined:

                                                             Figure 1: Spatial Model

 

The user should fulfill the data from 1 to 3, select wich components will be used, 9 to 11, the Biomass component (7) and the Deforest component (8) always has to be used, at least inform the Spatial Data clicking on Project Terraview (4a) or Shape (4b). 

 

If a Shape file is selected, the option Layer Name, 5, will not be avaliable.

 

The Cell size, 6, must be fullfiled with the same value used on the Demand calculation.

 

If the user want consoles outputs, should select Verbose, 12.

 

Non Spatial Model parametrization should be done when the user want to use a Non Spatial Model or a Combine Model.
