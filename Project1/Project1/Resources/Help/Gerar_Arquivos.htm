This section handles with the files generation (.lua).

Two files will be saved: one with the model name and _main.lua, having on it the configuration information of the model; a second one with the model name and _submodel.lua, having on it the selected components and its data.

Be sure to fulfill all the necessary data to make the files.

In order to make the files click on Make Files Button:

   

                                                                     Figure 1: Make Files Button    

 

After a success file generation a message will show:

                                                  

                                                              Figure 2: Files successfully recorded   

 

The Run Model Button will be visible after the files have been recorded, enabling to the user to execute the model that was saved.

                                                                  Figure 3: Run Model Button
