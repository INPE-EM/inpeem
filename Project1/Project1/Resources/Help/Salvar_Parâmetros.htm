This section handles with the model outputs.

Here is defined:

  • The output theme name, the theme name and the executed year will be combined to create the file name (e.g. Output Theme Name = output, Start Time = 1999, Final Time = 2000; the save themes will be: output1999 and output2000, in case of these years was selected),
  • The years to be save and
  • The land use types, that will be saved in three columns (e.g. Land Use = Forest; Forest_out, Forest_pot and Forest_change).

                                                                Figure 1: Parameters to Save

 

You have two options to select the years to be saved:

1.       Check the option Yearly Save.

 

                      

                                                                  Figure 2: Yearly Save

 

 

2.       Uncheck Yearly Save and select the specific years.

                         

                                                                        Figure 3: Years to Save

 

 

To do multiple selection of the years to be saved, just press Ctrl key and click on the desires years.

After click on Select button, a confirmation will appear:

 

                            

                                                          Figure 4: Years to Save Confirmation

 

To undo the years selection, just click on Select button without select any year.

The process to choose the Land Use Types, or Attributes, to be saved happens the same way.

To do multiple selection just press Ctrl key and click on the desires Attributes.

 

After that, click on Select button and a confirmation will appear:

 

                                                         

                                                Figure 4: Attributes to Save Confirmation
