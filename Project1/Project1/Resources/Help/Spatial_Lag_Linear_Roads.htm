This component uses the Land Use Types to input the data.

To select it just click on Spatial Lag Linear Roads.

 

                         Figure 1: Spatial Lag Linear Roads Button

 

 After that, the component window will appear:

                        Figure 2: Spatial Lag Linear Roads Window

 

To enter the component data you must select the Land Use Type that you want to enter the data; the window will change to the data input mode:

 

           Figure 3: Spatial Lag Linear Roads Window – Editing

 

Clicking on Roads Model button will cause the window to expand, making possible to enter the data for the road model of the selected Lan Use Type:

 

                                                         Figure 4: Spatial Lag Linear Road – Editing – Road Model

 

After enter the values of each land use the status will be modified on the main window of the component.

      Figure 5: Spatial Lag Linear Roads Window – Modified Status

 

 

After save the data a confirmation will appear:

                                     

          Figure 6: Potential – Spatial Lag Linear Roads Confirmation
