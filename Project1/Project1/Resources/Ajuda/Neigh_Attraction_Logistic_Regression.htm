Este componente utiliza os Tipos de Uso da Terra para que o usuário possa entrar com os seus dados.

Para selecionar este componente clique no botão Neigh Attraction Logistic Regression:

               Figura 1: Botão Neigh Attraction Logistic Regression

 

Após a seleção a janela do componente aparecerá:

              Figura 2: Janela Neigh Attraction Logistic Regression

 

Para configurar o componente é necessário selecionar o Tipo de Uso da Terra ao qual se desejar inserir os dados de regressão, a janela modificará para o modo de inserção de dados:

       Figura 3: Janela Neigh Attraction Logistic Regression - Inserção

 

Ao entrar com os valores de regressão para cada uso seu status é modificado na janela principal do componente:

 

      Figura 4: Neigh Attraction Logistic Regression - Status Modificado

 

Após salvar os dados a confirmação será apresentada na tela:

       

      Figura 5: Confirmação Potencial – Neigh Attraction Logistic Regression
