Nesta seção, lida com a parametrização do Modelo Espacial.

                                                             Figura 1: Modelo Espacial

 

 

O usuário deverá preencher os dados de 1 a 3, selecionar quais componentes serão utilizados, 8 a 11, o componente de Biomassa (7) sempre deverá ser utilizado, e informar os dados Espaciais clicando em Projeto Terraview (4a) ou Shape(4b).

 

Se um arquivo Shape for selecionado, a opção Nome do Layer, 5, não estará disponível.

 

O Tamanho da Célula, 6, deve ser preenchido com mesmo valor usado para o cálculo da Demanda.

 

Caso deseje ativar a saída no Console, deve-se selecionar Verboso, 12.

 

A parametrização do Modelo Espacial deve ser feita sempre que o usuário deseje gerar um Modelo Espacial ou um Modelo Combinado.
