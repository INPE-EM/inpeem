Este componente possui três parâmetros de configuração e uma matriz de transição de tamanho: número de tipos de uso X número de usos (ex.: Usos - F, D, O = matriz 3x3). Essa matriz deve ser preenchida somente com 0 (transição não permitida) ou 1 (transição permitida).

Para acessá-lo basta clicar em no Botão Allocation ClueS Like Neighbor Ordering.
