Para realizar uma validação ou calibração acesse a aba Validação:

 

                                                                      Figura 1: Janela Validação

 

Nesta versão duas metodologias estão disponíveis:

1.       Considerando todo o espaço celular (Ext)

2.       Considerando apenas as áreas modificadas (Dif)

Para executar uma validação os arquivos do modelo devem ter sido gerados (Gerar Arquivos) e executados (Rodar Modelos).

 

Se os arquivos foram gerados corretamente o botão Validar estará visível, bastando inserir os dados de validação:

1.       Escolha o método de Validação/Calibração.

2.       Nome do Tema – Nome do tema de saída do modelo executado.

3.       Atributo Inicial – Nome do atributo real inicial, considerado no primeiro ano de simulação (Ex. desmatamento).

4.       % Erro Permitido por Célula – Nível de tolerância de erro a ser considerado por célula (Ex. 5% - de 0 a 5% de erro é considerada uma célula correta).

5.       Atributo a ser Validado – Nome do atributo a ser validado gerado pela execução do modelo. Atributo selecionado em Salvar Parâmetros (Ex. desmatamento_out).

6.       Atributo Real Final – Nome do atributo real final, ano considerado o de validação.

7.       Número de Janela – Número de janelas considerado na validação, sendo 1 uma comparação célula a célula e números maiores que 1 um agrupamento de células (Ex. 10, serão feitas comparações de 1 janela até 10 janelas) .

8.       Salvar no Banco de Dados – Selecione se deseja salvar o mapa das diferenças em cada uma das janelas no banco de dados (Observação quanto aos valores: Zero - Correto; Negativos - Erro por comissão; Positivos - Erro por omissão).

 

                                                                 Figura 2: Dados para validação

 

Após informar os dados para validação clique no botão Validar e a janela de validação a parecerá:

 

             

                                                           Figura 3: Radando a Validação

 

Ao fechar a janela de validação, um arquivo .txt será salvo na pasta do modelo com os dados apresentados na tela.
