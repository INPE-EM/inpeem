Este componente utiliza os Tipos de Uso da Terra para que o usuário possa entrar com os seus dados.

Para selecionar este componente clique no botão Spatial Lag Linear Roads:

                             

                                                     Figura 1: Botão Spatial Lag Linear Roads

 

Após a seleção a janela do componente aparecerá:

                              

                                                                    Figura 2: Janela Spatial Lag Linear Roads

 

Para configurar o componente é necessário selecionar o Tipo de Uso da Terra ao qual se desejar inserir os dados de regressão, a janela modificará para o modo de inserção de dados:

                            

                                                     Figura 3: Janela Spatial Lag Linear Roads – Inserção

 

Ao clicar no Botão Roads Model o formulário ampliará, para ser possível a entrada dos dados referente ao Road Model do Uso da Terra selecionado:

              Figura 4: Spatial Lag Linear Road – Inserção – Road Model

 

Ao entrar com os valores de regressão para cada uso seu status é modificado na janela principal do componente:

                              

                                                        Figura 5: Spatial Lag Linear Roads - Status Modificado

 

Após salvar os dados a confirmação será apresentada ao usuário:

                                                        

                                                   Figura 6: Confirmação Potencial – Spatial Lag Linear Roads
